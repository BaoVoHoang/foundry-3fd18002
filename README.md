# GitLite

## One-Line Pitch
Empower your team with an intuitive, visual understanding of Git's inner workings.

## Features
- Interactive, visual Git concepts: Repositories, commits, branches, and three-way merges.
- Exposed storage model & algorithms for deep understanding.
- File Editor & Staging Area: Create, edit, delete files; track file status.
- Commit DAG & Content-Addressed Storage: Visualize Git history as a directed graph.
- Branch Management & Checkout: Create, list, switch branches; update HEAD.
- Visual Diff & Conflict Resolution: Compare changes with line-based diffs.

## Getting Started
1. Install dependencies: `npm install`
2. Run development: `npm run dev`
3. Build project: `npm run build`
